## 9 - Next steps

Congratulations! You've built a basic interactive game with Kubriko. This foundation covers the core loop of actors, traits, and managers that powers every Kubriko project.

### Extending Functionality

Because Kubriko is built on Kotlin Multiplatform, you can easily extend your game using the vast ecosystem of KMP libraries:
- **Networking**: Use [Ktor](https://ktor.io/) to add multiplayer capabilities or fetch remote content.
- **Serialization**: Use [Kotlinx Serialization](https://github.com/Kotlin/kotlinx.serialization) for complex save states (Kubriko's `persistence` plugin uses this under the hood).
- **Dependency Injection**: Use [Koin](https://insert-koin.io/) to manage your dependencies and game services.

### Where to go from here?

- **Explore the Examples**: The `/examples` folder in the repository contains everything from simple tests to full games like *Annoyed Penguins* and *Space Squadron*.
- **Read the Docs**: Check out the [List of Traits](https://github.com/pandulapeter/kubriko/blob/main/documentation/LIST_OF_TRAITS.md) and [List of Managers](https://github.com/pandulapeter/kubriko/blob/main/documentation/LIST_OF_MANAGERS.md) to see what other "superpowers" your actors can have.
- **Join the Community**: If you have questions or want to share your creations, join our Discord.
